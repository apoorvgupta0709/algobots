### Page 2 — Set Up -1
- Don’t open trade on specific time.
- RSI should be between range of 40-60 on 5 minute frame.
- ADX and DMI should be merge on 5 minute frame.
- ADX value should be below 25 on 5 minute frame.
- And always prefer ADX should be below both DMI.
- Nifty should between its day high and day low.
- Never create after 2:45 PM.

### Page 3 — Set Up -2
- Always wait for big fall.
- Stay away from straddle till you find any reversal.
- Wait for hammer or any strong candle pattern near good support.
- Wait for candle closing and initiate ATM straddle when trade initiate according to candle setup.
- Check India VIX; VIX should make a good reversal candle from top like shooting star or bearish engulfing.

### Page 4 — Set Up -2 continued
- Try to make ATM straddle, but if you have good conviction about reversal then you can go for upside strike straddle.
- If you don’t have conviction then you can go with downside strike straddle.
- Keep candle’s low as your exit point from trade.
- Keep shifting straddle as market moves upside.
- You can enjoy premium decay till some good resistance.

### Page 5 — Set Up -3
- Check spot at 9:30 AM.
- Wait for spike in this strike; if till 10:30 combined premium price goes above VWAP then wait.
- If combined premium comes below VWAP, sell the straddle and keep day high of combined premium as SL.
- After some time, trail SL according to VWAP.

### Page 6 — Adjustment - 1
- Adjust on 100 points move if premiums are above 350.
- Adjust on 75 points move if premiums are between 250-350.
- Adjust on 50 points move if premiums are between 150-250.
- Adjust on 25 points move if premiums are below 150.
- Always book ITM strike.
- Keep shifting from straddle to strangle and strangle to straddle.

### Page 7 — Adjustment - 1 alternate scale
- Adjust on 300 points move if premiums are above 1200.
- Adjust on 200 points move if premiums are between 800-1200.
- Adjust on 150 points move if premiums are between 600-800.
- Adjust on 100 points move if premiums are below 600.
- Always book ITM strike.
- Keep shifting from straddle to strangle and strangle to straddle.

### Page 8 — Adjustment - 2
- Make a straddle and wait till it breaches breakeven point.
- When it reaches BEP, book profit leg.
- Then sell same side ITM option in double quantity and sell one extra option of remaining leg.
- Do it continuously to increase range, but avoid more than 3 times.

### Page 9 — Strangle Set Up -1
- Use this as positional in weekly and monthly.
- Always try to find two support and two resistance levels.
- Try to sell beyond these two levels and collect at least ₹100.
- Adjust according to minor levels from both sides.
- Or adjust if one side premium becomes double.
- Shift loss leg if instrument breaks any major level.

### Page 10 — Strangle Set Up -2
- Use this setup for expiry.
- Find max OI strikes on both sides on Wednesday around 1 PM.
- Check support and resistance level near these strikes.
- Choose strikes outside of these strikes and levels.
- Choose premiums between ₹25-₹40.
- Whenever one side premium falls by 35-40%, shift leg to inside strike.
